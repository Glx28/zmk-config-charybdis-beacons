P2 - XbitLabs Mouse Accuracy (20% - MANDATORY)

OPEN TEST
SETTINGS:
- Difficulty: MEDIUM for all modes
- Same path type every time
Measures: Curved paths, shapes, steady hand control
4 Modes × 3 runs each = 12 total:
- Path Follow × 3
- Shape Trace × 3
- Steady Hand × 3
- Precision × 3
Record: Accuracy %, Deviation px, Failures
Why mandatory: Catches problems that don't show in straight-line tests (P1)

Total Testing Time

~75 minutes per profile:
- 3 min warmup
- R1: 5 min (3 runs)
- A1: 10 min (3 runs)
- A2: 15 min (5 runs)
- P1: 15 min (5 runs)
- P2: 30 min (12 runs total)

How to Use

1. Run PowerShell script: .\run_benchmark.ps1 -ProfileName "P0-current-3mon"
2. Script will guide you through each test
3. Open test links from this page
4. Enter results when prompted
5. Everything saves to benchmark_results.txt
6. Paste file to AI for analysis
